# selenium-bdd-framework
End-to-end Selenium automation framework built with Cucumber BDD, TestNG, Maven, Extent Reports and Word step-level reporting for scalable UI test automation.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Extent Reports
- Apache POI

## Features
- BDD test scenarios
- Step level screenshot capture
- Word execution report generation
- Maven profile based execution
- TestNG parallel execution

## How to Run

mvn clean test -PNormal
